# lic2020_baselines
some baselines for lic2020 (http://lic2020.cipsc.org.cn/)

## 依赖

```bash
pip install bert4keras==0.7.1
```

其他按需安装，建议`keras==2.3.1`、`tensorflow_gpu==1.14`，python2或python3均可。
